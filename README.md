# MadelbrotSetIamge

该程序可在py文件同目录下生成一张曼德博集合的图片
Will generate a nice image of Mandelbrot Set at the same directory of the .py file

使用的语言是Python3
Use Python3

需要名为“Pillow”的图形库
Needs "Pillow" libiary
https://github.com/python-pillow/Pillow
